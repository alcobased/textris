# Textris - A Polyomino Packing Game

This is a simple and interactive web-based puzzle game where the objective is to fit various polyomino shapes (like Tetris pieces) onto a grid. The game is built entirely with modern, framework-less web technologies.

## Features

*   **Interactive Grid:** A clickable grid that allows you to place and remove shapes.
*   **Shape Library:** A collection of standard polyomino shapes to choose from.
*   **Shape Manipulation:**
    *   **Rotate:** Turn shapes to find the best fit.
    *   **Flip:** Mirror shapes for more placement options.
*   **Placement Overlay:** See a preview of where the shape will be placed before you commit.
*   **Granular Clearing:**
    *   **Clear Shapes:** Remove all the placed shapes from the grid.
    *   **Clear Overlay:** Remove the current placement preview.
*   **Responsive Design:** The game interface is designed to work on various screen sizes.

## How to Play

1.  **Select a Shape:** Click on any shape from the library at the top.
2.  **Position the Shape:** Move your mouse over the grid to see a preview of where the shape will be placed.
3.  **Rotate or Flip:** Use the "Rotate" and "Flip" buttons to orient the shape as needed.
4.  **Place the Shape:** Click on the grid to place the shape in the desired location.
5.  **Clear the Board:** Use the "Clear Shapes" button to start over.

## Technologies Used

*   **HTML5:** For the core structure of the application.
*   **CSS3:** For modern styling, including CSS variables and a responsive grid layout.
*   **Modern JavaScript (ES6+):** For the game logic and interactivity.
*   **Web Components:** The main components of the game, like the grid and the control panel, are built as encapsulated, reusable custom elements.
